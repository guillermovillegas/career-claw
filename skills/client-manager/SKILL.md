# Client Manager

Track and manage freelance client engagements for Guillermo Villegas.

## Client Record Structure

Track each client with the following fields using the jobclaw tracker tool:

- **Client name** -- company or individual
- **Contact person** -- primary point of contact (name, email, role)
- **Status** -- lead / active / paused / completed
- **Project name** -- short descriptor of the engagement
- **Start date** -- when work began
- **Rate** -- hourly rate or fixed price (target range: $150-250/hr)
- **Scope summary** -- 2-3 sentence description of deliverables
- **Milestones** -- list of deliverables with target dates and status
- **Hours logged** -- running total of billable hours
- **Invoices** -- amounts, dates sent, payment status
- **Notes** -- communication log, key decisions, scope changes

## Client Lifecycle

### Lead

A potential client who has expressed interest or been pitched.

- Log initial contact and source (Upwork, referral, cold outreach, etc.)
- Record project requirements and budget
- Set follow-up reminder for 2-3 days
- Move to Active when contract/agreement is signed

### Active

Ongoing engagement with billable work.

- Track hours weekly
- Update milestone progress after each work session
- Log all client communications (decisions, feedback, approvals)
- Send invoices per agreed schedule (weekly, biweekly, or per milestone)
- Set weekly check-in reminder

### Paused

Work temporarily stopped (client request, waiting on input, scheduling gap).

- Log reason for pause
- Set follow-up reminder for 1 week
- Keep notes on where work left off for easy resumption

### Completed

Engagement finished.

- Confirm all deliverables accepted
- Send final invoice
- Log total hours and revenue
- Request testimonial or referral if relationship was strong
- Add to portfolio if applicable

## Milestone Tracking

For each project, break work into milestones:

```
Milestone: [Name]
Target date: [Date]
Status: not started / in progress / review / completed
Deliverables: [List of specific outputs]
Notes: [Any relevant context]
```

Update milestone status after every work session. Flag any milestones at risk of slipping and communicate proactively to the client.

## Communication Logging

Log every substantive client interaction:

```
Date: [Date]
Type: email / call / message
Summary: [1-2 sentence summary of what was discussed or decided]
Action items: [Any commitments made]
```

Use this log to:

- Prepare for client calls (review last 3-5 interactions)
- Resolve disputes about scope or agreements
- Track patterns in feedback

## Invoice Tracking

For each invoice:

```
Invoice #: [Number]
Date sent: [Date]
Amount: [Dollar amount]
Period: [Date range or milestone]
Hours: [If hourly]
Status: sent / paid / overdue
Payment date: [Date received, if paid]
```

**Invoice reminders:**

- If unpaid after 7 days: send a friendly reminder
- If unpaid after 14 days: send a firmer follow-up
- If unpaid after 30 days: flag for direct conversation
- Track total outstanding across all clients

## Scope Change Management

When a client requests work outside the original scope:

1. Acknowledge the request
2. Assess impact on timeline and budget
3. Present options: add to scope with revised timeline/cost, or defer to a follow-up phase
4. Get written approval before starting work on scope additions
5. Update milestones and rate if applicable

Never absorb significant scope creep for free. Small favors build goodwill; large unscoped work burns margin.

## Rate Negotiation

- Default hourly range: $150-250/hr depending on complexity and platform
- Fixed-price projects: estimate hours at the hourly rate, add 20% buffer
- Retainer arrangements: minimum 10 hours/month at a slight discount
- Rush work (< 48 hour turnaround): 1.5x standard rate
- If a client pushes back on rates: reduce scope before reducing rate

## Weekly Review Checklist

Run this every Monday:

1. Review all active clients -- any milestones due this week?
2. Check for overdue invoices -- send reminders
3. Review paused clients -- time to follow up?
4. Check leads pipeline -- any follow-ups needed?
5. Log total hours billed last week across all clients
6. Flag any clients with communication gaps (no contact in 7+ days)
