# Fiverr Search Reference

## Buyer Requests Navigation

### Accessing Buyer Requests

1. Navigate to `fiverr.com` and verify login
2. Go to seller dashboard: `fiverr.com/users/[username]/seller_dashboard`
3. Click "Buyer Requests" or navigate to `fiverr.com/users/[username]/seller_dashboard/briefs`
4. Filter by category matching active gigs

### Buyer Request Details

For each buyer request, extract:

1. **Description:** What the buyer needs
2. **Budget:** Price range the buyer is willing to pay
3. **Delivery time:** Expected deadline
4. **Buyer info:** Member since, country, previous orders
5. **Category/subcategory:** The service category
6. **Number of offers sent:** Competition level

### Filtering Strategy

Focus on requests matching these categories:

- Web Application Development
- AI & Machine Learning
- Full-Stack Development
- SaaS/Web Platform Development

Skip requests that:

- Have budgets under $200 for development work
- Require technologies outside Guillermo's stack
- Have unrealistic timelines (e.g., full SaaS in 2 days)
- Are from buyers with no previous orders and vague descriptions

## Fiverr Gig Search (Finding Demand Signals)

To understand market demand and position gigs effectively:

1. Navigate to `fiverr.com/search/gigs?query=QUERY`
2. Review top sellers in relevant categories
3. Note: pricing tiers, delivery times, order queues, review counts
4. Use this data to price and position Guillermo's gigs competitively

### Relevant Search Queries

- `next.js developer`
- `react typescript developer`
- `ai integration web app`
- `saas development supabase`
- `computer vision developer`
- `mvp prototype development`

## Responding to Buyer Requests

When responding to a buyer request:

1. Read the full request carefully
2. Match to Guillermo's relevant experience
3. Draft a concise offer (see `job-apply` skill for proposal templates)
4. Set appropriate pricing (reference `guillermo-profile` for rate guidelines)
5. Select the most relevant gig to attach to the offer
6. Submit the offer

## Gig Analytics

Check gig performance at `fiverr.com/users/[username]/seller_dashboard/analytics`:

- Impressions and clicks
- Conversion rate
- Average response time
- Order completion rate

Use analytics to identify which gigs need optimization (title, tags, description, pricing).
