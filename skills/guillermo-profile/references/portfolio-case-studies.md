# Portfolio Case Studies

Source: GuillermoTheEngineer.vercel.app

## 1. Levee

Award-winning AI platform that cut hotel inspection times by 60%. Computer vision at 92%+ accuracy, deployed across 10,000+ hotel rooms. Global Startup Pitch Winner 2025.

- **Tech:** React, Next.js, TypeScript, Python, YOLO/RT-DETR, GCP, Supabase
- **Metrics:** 60% faster inspections, 92%+ accuracy, $500K+ funding

## 2. SunrAI CRM

Complete solar business platform from lead capture to project completion. AI-powered proposal generation via Google Solar API and Gemini. Multi-tenant with RLS.

- **Tech:** Next.js, React, TypeScript, Supabase, Google Gemini, Google Solar API
- **Metrics:** 70% faster workflows, sub-2s page loads, 99.9% uptime

## 3. SunrAI Admin

Enterprise admin dashboard with real-time AI cost tracking, 5-tier user management, and feature flags. Powers multi-company operations.

- **Tech:** Next.js, React, TypeScript, Supabase, Recharts
- **Metrics:** 70%+ performance optimization, 5 permission tiers

## 4. APACT Trading

Enterprise OTC trading platform for energy commodity brokers. Complete trade lifecycle with 4-tier access control and audit compliance.

- **Tech:** Next.js, TypeScript, Supabase, PostgreSQL
- **Metrics:** 46 API endpoints, real-time analytics, enterprise security

## 5. SoCap Solar

Multi-tenant solar financing SaaS with 3 apps. Milestone-based workflow, Figure API integration.

- **Tech:** Next.js, React, TypeScript, Supabase
- **Metrics:** 204 React components, 15,758 lines server actions, $1M+ transactions

## 6. SunrAI Mobile

Field app for solar technicians with offline-first architecture. AI photo categorization, GPS tracking.

- **Tech:** React Native, Expo, TypeScript, Supabase
- **Metrics:** 99%+ reliability, offline-first

## 7. Remo AI

AI-native project management using Gemini for intent inference and dynamic UI generation.

- **Tech:** Next.js, TypeScript, Google Gemini, Supabase
- **Metrics:** Intent-based AI, dynamic UI generation

## 8. Screenshotter
